# ciclos

los ciclos en python son los mas sencillos de aprender solo hay dos ciclos estos son los siguientes:

``` python
# for
arr = [1,2]

for i in arr:
    print(i)
    
for i in range(5):
    print(i)

# while
opc = ''

while opc != 'quit':
    opc = input("Ingrese una opcion: ")
    print(f"Usted eligio: {opc}")
```
